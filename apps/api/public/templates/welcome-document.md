## 请尝试以下基本操作：

- 点击页面中任意位置，开始输入内容

- 点击左侧的 **+** 按钮或用键盘输入 **/** ，可以插入各种类型的内容或创建子页面

- 选中一段文字，通过弹出的菜单更改你喜欢的文字样式，或给文字增加网页超链接

- 拖拽左侧的 **⋮⋮** 按钮就可以移动位置，点击 **⋮⋮** 按钮则会显示菜单

- 点击左侧栏目录中 **+** 按钮，可以创建新页面

- 点击左侧栏中的 **模板中心** ，可以从预制的模板库中选择，快速开始使用

> 如果在使用中遇到任何问题，欢迎 [github issue 反馈](https://github.com/chenxiaoyao6228/idea-forge/issues/new),
